ℹ️ About Passle

What is Passle?

Passle is a set of daily guessing games built around XBOX Game Pass — past and present. Guess a game from its box art, a blurred screenshot, a cryptic riddle, real player reviews, or name which Game Pass plans it lives on. There's also an OG XBOX mode for backward-compatible classics.

Pick Daily for one shared puzzle a day, or Endless to keep playing. Tune the catalog to the plans you actually own in Settings.

💡 Inspired by

Passle stands on the shoulders of the guessing games that made the format fun:

  • Gamedle — daily "guess the game" puzzles (box art, screenshots, higher/lower, and more).
  • GuessThe.Game — guess the game from a series of screenshots.

Passle is an independent fan project and is not affiliated with, endorsed by, or connected to either of them.

🔧 Built with

  • Vanilla web — plain HTML, CSS, and JavaScript, no frameworks or build step. It ships as a static site, installable as a PWA on phones.
  • Python — small scripts assemble and enrich the game data (catalog fetch, IGDB enrichment, image resolution), and a tiny stdlib server (serve.py) hosts the site plus an opt-in leaderboard API.
  • Local-first — your stats and settings live in your browser's local storage. The shared leaderboard is opt-in: a score only leaves your device if you turn on sharing, and only an anonymous id (or, in Teams, your name) is stored.
  • Microsoft Teams — Passle runs as a Teams personal app; the Microsoft Teams JavaScript SDK supplies your Microsoft 365 identity so the daily leaderboard is tied to your team.

📡 Data & sources

  • XBOX public catalog & Microsoft Store — game titles, box art, screenshots, player reviews, and Game Pass plan availability for modern titles.
  • IGDB — cover art, screenshots, and critic/user scores, especially for the OG XBOX library. Game data provided by IGDB / powered by IGDB.
  • Wikipedia & Wikidata — supplementary metadata (developers, publishers, genres, release dates) for classic titles.

Catalog data is refreshed at build time; nothing is fetched from these sources while you play.

🔒 Privacy & data

Passle keeps things light. Almost everything lives on your device:

  • Your profile, settings, stats and streaks are stored only in this browser's localStorage — there are no accounts and no login.
  • Anonymous usage count — when Passle is being hosted, each page load sends a tiny, anonymous ping (a random per-device id and the page name, no personal info) so the host can see roughly how many people are playing.
  • Shared leaderboard (opt-in) — your gamerscore is posted to the host so everyone playing the same hosted Passle shares one board. You appear as “Anonymous” unless you turn on “Show my name on the shared leaderboard” in your profile, in which case your Passle name is visible to other players.

Opened as a plain file with no server, the ping and leaderboard simply do nothing — your stats stay entirely local.
